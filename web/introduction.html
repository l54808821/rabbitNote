# 介绍

[Rabbit]是一个开源的自动化测试平台，基于经典技术组合(Spring Boot、Spring Security、MyBatis、Jwt、Vue)，目前版本已支持UI自动化测试。平台采用关键字驱动，测试人员无需任何代码基础，即可轻松完成复杂的自动化测试工作。
